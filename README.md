# Tools UMKM (Bug Bounty & Recon Environment)

Script installer otomatis buat nyiapin environment recon dan bug bounty di Kali Linux atau VPS dengan cepat dan anti ribet.

## Overview

Tools-umkm ini dirancang modular buat bantu kamu nyiapin semua kebutuhan tools pentest dan recon dalam hitungan menit tanpa harus konfig manual satu-satu.

## Quick Installation

Tinggal jalankan perintah ini di terminal:

```bash
git clone [https://github.com/bobriva/tools-umkm.git](https://github.com/bobriva/tools-umkm.git)
cd tools-umkm
sudo ./install.sh
